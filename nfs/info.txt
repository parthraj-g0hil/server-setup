On NFS-Server

Install NFS Kernel Server in Ubuntu
        sudo apt install nfs-kernel-server

Create an NFS Export Directory
        sudo mkdir -p /mnt/nfs_share
        sudo chown -R nobody:nogroup /mnt/nfs_share/
        sudo chmod 777 /mnt/nfs_share/
        
Grant NFS Share Access to Client Systems (You can provide access to a single client, multiple clients, or specify an entire subnet.)
        vim /etc/exports
                /mnt/nfs_share  16.0.0.0/16(rw,sync,no_subtree_check)
                
Export the NFS Share Directory
        sudo exportfs -a




Install the NFS Client on the Client Systems

install
        sudo apt install nfs-common
 
 Create an NFS Mount Point on Client
        mkdir -p /mnt/nfs_clientshare

Mount NFS Share on Client System
        mount 10.60.3.222(nfs-server):/mnt/nfs-share  /root/partrhraj
